## üîß COMMON FIXES FOR CHARACTER INPUTS

### If a character won't attack:

1. **Check Statedef -4** ‚Üí MUGEN doesn't support it! Move to Statedef -3
2. **Check var() initialization** ‚Üí Move enable variables must be set to 1
3. **Check command triggers** ‚Üí Every State -1 move needs `command = "..."`
4. **Check AI activation** ‚Üí `var(63) = 1` when `AILevel > 0`

### Adding Commands to CMD File:

```ini
; Quarter Circle Forward + Punch
[Command]
name = "QCF_x"
command = ~D, DF, F, x
time = 20

; Dragon Punch Motion
[Command]
name = "DP_x"
command = ~F, D, DF, x
time = 20

; Charge Move (hold back 60 frames, then forward + kick)
[Command]
name = "charge_b_f_a"
command = ~60$B, F, a
time = 20
```

### Adding Command Triggers to Moves:

```ini
[State -1, Soul Fist]
type = changestate
value = 1000
triggerall = var(71)  ; Move enable flag
triggerall = command = "QCF_x" || command = "QCF_y" || command = "QCF_z"  ; INPUT!
triggerall = roundstate = 2 && statetype != A
trigger1 = ctrl
```
